# 🏰 Tower Defense

> Мобильная стратегия в жанре Tower Defense, разработанная на Unity.

---

## 📖 Описание игры

**Tower Defense** — это стратегическая игра, в которой игрок защищает главную базу от волн наступающих врагов. На каждом уровне враги движутся по заранее заданному маршруту и пытаются уничтожить базу. Задача игрока — грамотно расставить башни на игровой сетке, чтобы остановить все волны.

### Геймплей

- 🗺️ **Сетка размещения** — башни устанавливаются на клетчатом поле. Каждая башня занимает определённое количество клеток (1×1, 2×2, 2×3 и т. д.)
- 🔫 **Обычная башня** — самонаводящийся снаряд, который следит за ближайшим врагом, поворачивается к нему и стреляет с перезарядкой
- ⚡ **Башня-аннигилятор** — работает циклами: Зарядка → Стрельба огнемётом → Перезарядка. Наносит урон всем врагам в зоне огня одновременно
- 👾 **Враги** — движутся по точкам пути (waypoints), имеют здоровье, сопротивление урону и скорость. Появляются волнами; с каждой волной состав врагов меняется
- 🏠 **Главная база** — если враги до неё добираются, они начинают атаковать; при разрушении базы игра заканчивается
- 💰 **Экономика** — за уничтожение врагов начисляются монеты, которые тратятся на покупку новых башен
- 🔄 **Перемещение башен** — уже установленную башню можно выбрать, переместить и повернуть на новое место

### Управление

| Платформа | Действие | Способ |
|-----------|----------|--------|
| ПК | Перемещение камеры | ПКМ + перетаскивание / WASD / края экрана |
| ПК | Зум камеры | Колесо мыши |
| ПК | Выбор башни | ЛКМ по башне |
| Мобильный | Перемещение камеры | Один палец — свайп |
| Мобильный | Зум камеры | Два пальца — пинч |
| Мобильный | Выбор башни | Тап по башне |
| Оба | Установка башни | Перетаскивание на нужную клетку → кнопка ✓ |
| Оба | Поворот башни | Кнопка поворота в панели подтверждения |
| Оба | Отмена | Кнопка ✗ / клавиша Escape |

## 🚀 Запуск проекта

### Требования

- **Unity** версии **6000.4 LTS** или новее
- Модули редактора:
  - `Android Build Support` — для сборки на Android
  - `iOS Build Support` — для сборки на iOS (опционально)

### Сборка для Android

1. **File → Build Settings**
2. Выберите платформу **Android**, нажмите **Switch Platform**
3. Откройте **Player Settings** и укажите:
   - `Package Name` — например `com.yourname.towerdefense`
   - `Minimum API Level` — Android 7.0 (API 24) или выше
4. Подключите устройство по USB (включите **Отладку по USB** в настройках разработчика) или выберите **Build** для создания APK-файла
5. Нажмите **Build And Run**

> ⚠️ Для корректной работы сенсорного управления в настройках проекта (**Edit → Project Settings → Player**) убедитесь, что активен пункт **Active Input Handling: Both**.

## 🛠️ Использованные технологии

### Движок и язык

| Технология | Версия | Назначение |
|------------|--------|------------|
| [Unity](https://unity.com/) | 6000.4 | Игровой движок |
| C# | 10 (.NET Standard 2.1) | Язык программирования |

### Unity-пакеты

| Пакет | Назначение |
|-------|------------|
| **Unity Input System** | Унифицированная обработка ввода (мышь, клавиатура, тач) |
| **Unity UI (uGUI)** | Элементы интерфейса: кнопки, слайдеры, панели |
| **TextMeshPro** | Высококачественный текст в UI |
| **Cinemachine** *(опционально)* | Управление камерой |
| **Universal Render Pipeline (URP)** *(опционально)* | Визуальный рендеринг |

### Архитектурные паттерны

| Паттерн | Где используется |
|---------|-----------------|
| **Service Locator** | Глобальный доступ к сервисам (`SoundFeedback`, `MainBase`, `EndGameManager`) без прямых ссылок |
| **State Machine** | Система размещения башен (`PlacementState`, `TowerMoveState`, `RemovingState`) |
| **Singleton** | `EndGameManager`, `PlacementSystem`, `TowerMoveUI` |
| **ScriptableObject** | Базы данных башен (`TowersDatabaseSO`) и врагов (`EnemiesDatabaseSO`) |
| **Observer / Events** | Реакция на гибель базы, смену волны, уничтожение врага |
| **Coroutine** | Цикл аннигилятора, анимации UI, асинхронная загрузка сцен |
